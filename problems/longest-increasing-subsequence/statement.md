# Longest Increasing Subsequence

You are given `n` integers `a[1..n]`. Find the length of the longest **strictly** increasing
subsequence — a subsequence where each element is greater than the one before it. Elements do
not need to be contiguous.

## Input

- Line 1: one integer `n` (`1 <= n <= 100000`).
- Line 2: `n` integers `a[1], ..., a[n]` (`-10^9 <= a[k] <= 10^9`).

## Output

A single line containing the length of the longest strictly increasing subsequence.
